📌 计算规则：选择/输入型号 → 自动填入最低成交价 + 国补下限价（红色锁定） → 原价=最低成交价（锁定）

⭐ 国补规则：有国补产品 → 折扣价 = 最低成交价×折扣系数，钳位到[国补下限价, 国补上限价]；触碰边界则锁定显示

💙 特价产品：11款特价产品 → 最低成交价列显示特价（蓝色锁定），各折扣列 = 特价（锁定），不参与整单基准与分摊（分子分母均剔除）

🔢 无国补分摊：整单折扣基准 = 系数×非特价最低成交价总和 → 减国补钳位 → 按普通产品最低成交价占比分摊

💰 客户实付：实付（原价）=原价直接相加；其余三列：无国补→实付=该列价格；有国补→基于该列折扣价×0.85，若折扣价×0.15≥1500则封顶减1500

🎁 冰箱红包：选中508冰箱型号即享激励（产品红包+店长红包），无需折扣条件

🎁 6月政策红包：选中对应型号按整单平均折扣分档激励（≥0.8/≥0.75/<0.75）

🎁 组合奖励：JBS2T-X20Max/L + 任意洗碗机 → 按整单平均折扣分档（≥0.8→¥300 / ≥0.75→¥150 / <0.75→¥0）

🔍 型号输入：输入任意字符实时筛选匹配 → 点击/回车选中；键盘上下键切换，Esc关闭
